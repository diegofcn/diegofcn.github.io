Project Overview

Jarvis is a virtual assistant like Alexa or Siri. It is based on the movie Iron Man. I built this program as a final project for Harvards CS50: Introduction to Computer Science course. This course teaches students how to think algorithmically and solve problems efficiently. Topics include abstraction, algorithms, data structures, encapsulation, resource management, security, and software engineering. Languages include C, Python, and SQL plus students choice of: HTML, CSS, and JavaScript (for web development).

Somme tasks Jarvis can do for you: Tell you the current date/time, play a song, open programs/websites, get the latest news, tell you random jokes/facts, empty recylce bin, write and show notes, get locations, restart your PC.
